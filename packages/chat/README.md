# Chat Package

This package is responsible for handling chat message intent classification, processing, and orchestrating AI-powered responses for the Edit Mind video analysis system.

## Features

- **Intent Classification**: Automatically classifies user prompts into different intent types (compilation, refinement, similarity, analytics, general)
- **Multi-Modal Search**: Processes search queries across text, audio, and visual content
- **Scene Search**: Search video scenes based on natural language queries
- **Analytics Generation**: Provides insights and analytics about video libraries
- **Similarity Search**: Finds scenes similar to previously selected content
- **Conversation Context**: Maintains chat history for context-aware responses
- **Project Scoping**: Supports filtering results by project-specific videos
