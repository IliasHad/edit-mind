# UI Package

This package is a foundational component library for the project, intended to house reusable UI components that can be shared across the web and desktop applications. It ensures consistency and accelerates development.

## Features

- **Reusable UI Components:** Provides a centralized place for common UI elements, promoting consistency in design and functionality across different applications.
- **Cross-Platform Compatibility:** Designed to be used in both the web (`apps/web`) and desktop (`apps/desktop`) applications.
